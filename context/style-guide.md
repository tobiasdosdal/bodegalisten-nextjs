# Bodegalisten Style Guide

## Brand Identity

Bodegalisten is a Copenhagen-focused bar discovery app with a warm, tavern-inspired aesthetic.

## Color Palette

### Primary Colors
- **Amber/Gold**: Primary brand color - warm, inviting tavern feel
- **Dark Brown**: Secondary - rich, earthy tones
- **Cream/Off-white**: Background - warm and approachable

### Semantic Colors
- **Success**: Green (`#22c55e`)
- **Error**: Red (`#ef4444`)
- **Warning**: Amber (`#f59e0b`)
- **Info**: Blue (`#3b82f6`)

### Dark Mode
- Background: Deep charcoal/slate tones
- Text: Light cream/off-white
- Accents: Muted gold/amber

## Typography

### Font Family
- Primary: System sans-serif stack
- Headings: Consider a slightly warmer, friendlier typeface

### Scale
- H1: 2rem (32px)
- H2: 1.5rem (24px)
- H3: 1.25rem (20px)
- Body: 1rem (16px)
- Small/Caption: 0.875rem (14px)

## Spacing

Base unit: 4px
Scale: 4, 8, 12, 16, 24, 32, 48, 64px

## Border Radius

- Small (inputs, buttons): 6px
- Medium (cards): 8px
- Large (modals): 12px
- Full (avatars, pills): 9999px

## Components

### Buttons
- Primary: Amber/gold background, dark text
- Secondary: Outlined, dark border
- Destructive: Red background
- Ghost: Transparent with hover state

### Cards
- Subtle shadow
- Rounded corners (8px)
- Warm background tint

### Map Markers
- Custom pin design reflecting bar/venue type
- Clear visual hierarchy for selected states
- Accessible contrast ratios

## Iconography

Use Lucide icons for consistency:
- Consistent stroke width
- Rounded line caps
- Clear, recognizable shapes

## Interaction States

### Hover
- Subtle background color shift
- Smooth transition (150ms)

### Focus
- Clear outline ring
- High contrast for accessibility

### Active/Pressed
- Slightly darker/pressed appearance

### Disabled
- Reduced opacity (50%)
- No pointer events

## Responsive Breakpoints

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## Accessibility

- WCAG AA minimum contrast ratios
- Keyboard navigable
- Clear focus indicators
- Semantic HTML structure
- ARIA labels where needed
